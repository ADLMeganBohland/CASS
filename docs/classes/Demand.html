Demand Class

Defined in: org.schema.js:4866
Module: schema.org

Schema.org/Demand A demand entity represents the public, not necessarily binding, not necessarily exclusive, announcement by an organization or person to seek a certain type of goods or services. For describing demand using this type, the very same properties used for Offer apply.
